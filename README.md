# 🌦️ Weather Fetch API

A lightweight application that fetches real-time weather data for any city using a public weather API. This project demonstrates API integration, asynchronous operations, and basic error handling.

---

## Features

- Fetch current weather by city name  
- Displays temperature, humidity, and weather conditions  
- Handles invalid inputs and API errors  
- Fast and simple implementation  

---

##Tech Stack

- JavaScript (Node.js)  
- Fetch API / Axios  
- OpenWeather API
